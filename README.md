# 蓝牙快速配置助手  
---------  
Overview:
* 本程序可以用来快速配置蓝牙模块，即使不知道模块的波特率。
* github 地址： <https://github.com/JamesMurrayBIT/BluetoothFastConfig>  
---------  
帮助：  
* 在两个模块没有配对上（指示灯闪烁）时，进行配置。  
* 只需要将主从模块的密码和波特率配置相同，再次上电即可自动配对。  
* 主模块不能配置名字（配置主模块名字会失败）但不影响模块正常使用。
